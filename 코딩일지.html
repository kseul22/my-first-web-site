TAG

  1. storng은 진하게 나타내는 강조 표시

  2. u는 밑줄 표시

  3. H1~H6은 제목을 나타내는 태그

  4. 줄바꿈 태그[html new ling tag]-br

    5. br은 줄바꿈이라고 하는 시각적인 의미만 가지고 있기때문에 태그를 닫을 필요가 없다.

  5. [html paragraph tag]- p

    6. p 태그는 단락을 나타내는 태그이기 때문에 열리는 태그와 닫히는 태그가 필요하다.

  6. img은 이미지를 넣는 태그 (사이즈 조절 가능)

    7. img src(source)="이미지 파일명.jpg" width= "size" 속성(Attribute)Tag로 설명이 부족할 때 속성값을 넣어줌 (src,width의 순서위치는 상관없음) 이미지는 내 html 폴더 안에 넣은 후 이미지 이름 입력

  7. li는 목차(LIST)를 나타내는 태그

  8. LI 태그의 부모 UL
    -항목을 구분짓기 위한 태그(이때 같은 항목끼리 TAB키를 누르면 들여쓰기 가능)

  9. OL[ORDERED LIST]-순서 구분 목차 / UL[UNORDERED LIST]-순서 없는 목차

    10. [li] 리스트를 만들 때 부모 tag를 사용하여 어디서~어디까지 연관된 내용인지 구분한다. Tag에는 부모-자식 Tag가 있고, 포함된 관계로 연관될 때 사용한다. (ex) ul-li // ol-li
